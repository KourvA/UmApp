# UmApp
It's just an Kivy app for Andoid with logo and lable

# Install
+ clone
```bash
git clone https://github.com/Kourva/UmApp && cd UmApp
```
+ requirements
```bash
pip install kivy
```
+ run
```bash
python3 main.py
```
